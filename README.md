# 学生知识成果展示与交流+ssm

## 前言

本项目致力于为学生提供一个知识成果展示与交流的平台，通过此平台，学生可以互相展示自己的学术成果，交流学习经验，共同进步。项目采用目前流行的SSM框架进行开发，具备较高的稳定性与可扩展性。

## 内容介绍

学生知识成果展示与交流平台主要包括以下功能模块：

1. 用户注册与登录：方便学生进行身份认证，保护个人信息。
2. 成果展示：学生可以发布自己的学术成果，如论文、专利、项目等。
3. 交流区：学生可以在交流区提问、回答问题，分享学习经验。
4. 个人中心：学生可以查看自己的成果、提问及回答记录，方便进行个人管理。

## 技术介绍

- 语言：Java
- 使用框架：Spring、SpringMVC、MyBatis、微信小程序
- 前端技术：JS、Vue、CSS3、Uniapp
- 开发工具：IDEA/Eclipse、Uniapp
- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpstudy/Navicat
- JDK版本：jdk1.8
- Maven：apache-maven 3.8.1-bin
- 前端环境：Node.Js 12/14/16
